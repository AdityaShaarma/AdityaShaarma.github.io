Homework 5: Bigfoot Sightings Visualization

Visualization 1: Bigfoot Sightings by State

This bar chart visualizes the total number of reported Bigfoot sightings per U.S. state. It provides an overview of which states have the highest frequency of reports.

Encodings: The x-axis encodes state as a nominal variable, and the y-axis encodes the count of records as a quantitative variable (count():Q). Each bar’s color is also mapped to state using a categorical colormap, which helps visually separate each bar and prevents monotony.

Color map: The default categorical color scheme provides high contrast and evenly distributed hues across states, helping viewers distinguish between bars. Since the data represents discrete categories (states), a continuous colormap would not be appropriate here.

Transformations: I used Vega-Lite’s built-in aggregation count() instead of manually grouping in pandas, allowing Altair to automatically compute total counts per state. This simple aggregation was enough because we only needed state-level counts.

Visualization 2: Interactive Sightings per Year (Selected State)

This plot shows how Bigfoot sightings in a specific state have changed over time. A dropdown menu allows viewers to choose a state, making the visualization interactive and focused on a single region at a time.

Encodings: The x-axis encodes year as a temporal variable, and the y-axis encodes count():Q. The color channel maps each point to its state for clarity, and the size encoding is conditioned on the selected state using alt.condition(sel, alt.value(100), alt.value(20)). This highlights the selected state visually.

Color map: A categorical colormap is again used because each state is a distinct category. When a single state is selected, its color remains consistent across all years, maintaining interpretability.

Transformations: In the Python notebook, I extracted the year from the date column using pandas (df['year'] = df['date'].dt.year) and grouped the data by state and year to prepare it for visualization.

Interactivity: The dropdown selection parameter (alt.selection_point) lets the viewer choose a specific state to highlight. This improves clarity and engagement, allowing exploration of year-by-year changes while filtering out noise from other states.
